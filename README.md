# Kaspi.kz - Офлайн версия

Это офлайн версия сайта Kaspi.kz, адаптированная для работы без интернета.

## Что работает офлайн:
- ✅ Главная страница (index.html)
- ✅ Госуслуги (gosu.html)
- ❌ Магазин (magaz.html) - требует интернет для полной работы

## Как использовать на телефоне:

### Вариант 1: Через GitHub Pages (рекомендуется)

1. **Создайте репозиторий на GitHub:**
   - Зайдите на https://github.com
   - Нажмите "New repository"
   - Назовите репозиторий (например, `kaspi-offline`)
   - Сделайте репозиторий публичным
   - Нажмите "Create repository"

2. **Загрузите файлы:**
   - На странице репозитория нажмите "uploading an existing file"
   - Перетащите все файлы из папки `c:\Users\Admin\Documents\Fake` (кроме .gitignore и скриптов .ps1)
   - Нажмите "Commit changes"

3. **Включите GitHub Pages:**
   - Зайдите в Settings > Pages
   - В разделе "Source" выберите "Deploy from a branch"
   - Выберите ветку `main` и папку `/ (root)`
   - Нажмите "Save"
   - Подождите 1-2 минуты, сайт будет доступен по адресу `https://ваше-имя.github.io/kaspi-offline/`

### Вариант 2: Прямое открытие на телефоне

1. **Перенесите файлы на телефон:**
   - Скопируйте все файлы из папки на телефон
   - Откройте `index.html` в браузере телефона

2. **Установка как приложение (PWA):**
   - В браузере Chrome (Android) или Safari (iOS)
   - Нажмите "Добавить на главный экран" или "Поделиться"
   - Выберите "Добавить на главный экран"
   - Теперь сайт будет работать как приложение

## Структура файлов:
```
├── index.html          # Главная страница
├── gosu.html           # Госуслуги
├── magaz.html          # Магазин (требует интернет)
├── assets/             # Изображения и иконки
├── main/               # Основные ресурсы
└── manifest.json       # PWA манифест
```

## Примечания:
- Сайт использует локальные ресурсы, поэтому интернет не требуется для основных функций
- Магазин (magaz.html) - это сложная страница с внешними ресурсами, для полной работы требуется интернет
- Для установки как PWA приложения на телефон рекомендуется использовать GitHub Pages
